The BadSoft Mass Renamer is a program designed to batch rename groups of files, allowing the user to specify the renaming pattern.

Introduction
The BadSoft Mass Renamer was originally created because, as a computer user, I knew that there comes a time in everybody's life where they have to rename a large group of files. I looked around, and found a few programs that did the job. But they were either too difficult to use (Directory Opus), too weak (some VB project I found a while ago), or just plain expensive (Adobe Photoshop).
This program has been created around Adobe Photoshop 7's excellent batch renaming system, which allows you to include various fields in the new file names.

How to use
Simply run the program from Explorer, and you should eventually see the following screen:

By selecting a folder in the left-hand pane, a list of all files within that folder will be displayed on the right. You can click on the column headers to sort by criteria.

Only files with a tick mark next to them will be included in the list of files to rename. To tick single items, click in the black box next to the name. You can also use the toolbar underneath to select all files, invert the selection or deselect all files respectively.

The Renaming format group allows you to choose the format for the new filenames. In each of the boxes you can either type in literal text, select a field (curly brackets), or leave it blank.

To see the effects of your field selection, make sure the Test mode checkbox is ticked. When you press the Rename file(s) button, you will see a list of the selected files, along with the new names. No changes are made while the Test mode checkbox is checked.

When using serial fields, you can choose which number they start from by changing the box marked Serial number starts at. The order the files are listed in the right-hand pane are the order the files are given serial values.

The current date (ie. today's date) is used when using date fields.
